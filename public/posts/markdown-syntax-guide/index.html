12. HTML 직접 사용 (Raw HTML)#

Hugo는 기본적으로 마크다운 파일 내의 HTML 사용을 허용합니다 (markup.goldmark.renderer.unsafe = true 설정 시, 기본값은 false). PaperMod 테마나 Hugo의 기본 설정에 따라 동작이 다를 수 있습니다. 보안상 주의가 필요합니다.

13. 이모지 (Emoji)#

PaperMod 테마는 이모지 사용을 지원할 수 있습니다 (enableEmoji 파라미터, hugo.toml에서 설정).

1

:joy: :heart: :rocket:

:joy: :heart: :rocket:

(만약 enableEmoji = true 설정이 필요하다면 hugo.toml 파일의 [params] 섹션 또는 최상단에 추가해야 합니다.)

이것으로 마크다운 구문 가이드를 마칩니다! 이 외에도 다양한 확장 구문이나 테마별 지원 기능이 있을 수 있습니다.

  2. 이것은 또 다른 각주의 내용입니다. 긴 설명도 가능하며, 여러 문단으로 구성될 수도 있습니다.
